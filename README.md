# Coordina

Compose, configure, and deploy teams of OpenClaw AI agents to Kubernetes — no YAML required.

## Prerequisites

- Node.js 20+
- npm 10+
- macOS (Electron desktop app)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Type Checking

```bash
npm run typecheck
```

## Testing

```bash
# Run all tests once
npm run test

# Run tests with interactive UI
npm run test:ui
```

## Build

```bash
# macOS app bundle
npm run build:mac

# Linux
npm run build:linux
```

## Lint & Format

```bash
npm run lint
npm run format
```
